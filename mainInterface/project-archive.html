Project: SakShell
Tools Used: C++17, Visual Studio, Ubuntu, Git
SakShell is a custom Unix-like shell written in C++, offering a minimal yet powerful command-line interface for executing commands and managing system operations.
Key Features:
✅ Command Execution – Supports built-in commands and runs external executables
✅ Custom Built-in Commands – Includes echo, pwd, cd, and type for efficient system interaction
✅ Quoted Text Handling – Proper parsing of quotes and escape sequences
✅ Error Handling – Meaningful error messages for incorrect commands
✅ System Integration – Supports piping and interaction with common Unix utilities
✅ Interactive Shell – Provides a continuous prompt for seamless user experience
